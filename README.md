# Employee Data Analysis System

This project performs data analysis on employee records stored in an SQL (SQLite) database using Python.

## Technologies Used
- Python
- SQLite
- Matplotlib

## Analysis Performed
- Total number of employees
- Average salary calculation
- Department-wise employee distribution
- Data visualization using Matplotlib bar chart

## How to Run
1. Install Python
2. Install required library:
   python -m pip install matplotlib
3. Run the program:
   python main.py

## Notes
- Employee data is stored locally using SQLite.
- The database file is not uploaded to GitHub and can be recreated using the program.
